# Numbers Fact

a React-App to show facts about entered number

![alt text](https://github.com/mhdalmasri/Numbers-fact/blob/master/numbers.png)

## Installing

```
npm i
```

Then

```
npm start
```

## Built With

* Reactjs
* Redux
* Redux Saga
* Axios
* Github Pages
